# Setup Guide

## Prerequisites
- Node.js 18.17+ or 20+
- npm 9+

## Installation
1. Install dependencies:
   - `npm install`
2. Start development server:
   - `npm run dev`
3. Open:
   - `http://localhost:3000`

## Quality Checks
- Lint: `npm run lint`
- Type check: `npm run typecheck`
- Build: `npm run build`
